Ручное управление транзакциями

Задача
Напишите скрипт для выполнения следующих шагов:
  1. Начните новую транзакцию
  2. Добавьте в таблицу еще одного сотрудника
    id last_name first_name userid salary
    5 Ropeburn Audrey aropebur 1550
  3. Отметьте промежуточную точку транзакции
  4. Удалите все данные из таблицы
  5. Проверьте содержимое таблицы
  6. Откатите удаление данных. Добавленный сотрудник должен остаться.
  7. Проверьте содержимое таблицы
  8. Зафиксируйте изменения
